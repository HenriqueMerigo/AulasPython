# AulasPython

Comandos git:
    
    git pull (puxa todos os arquivos do git para o diretório)
    
    git add . (Adiciona todos os arquivos modificados no commit)

    git commit -m "mensagem" (Atualiza uma "maleta" de alteracoes para o projeto do git)

    git push (envia a "maleta" para o git a fim de que os membros do projeto possam fazer o pull atualizado)
